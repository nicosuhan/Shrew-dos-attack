Shrew DoS Attack

Un atac subtil, dar devastator pentru traficul TCP

🧠 Ce este Shrew?

Shrew este un atac de tip Denial-of-Service cu rată redusă, care nu suprasolicită rețeaua, ci o sabotează strategic, profitând de mecanismele de control al congestiei TCP.

⚙️ Cum funcționează?

  • Exploatează timeout-ul TCP (RTO) prin rafale scurte
  • Reduce dramatic performanța fluxurilor legitime
  • Este greu de detectat – nu generează trafic masiv
